AngularJS Controllers

AngularJS applications are controlled by controllers.

The ng-controller directive defines the application controller. A controller is a JavaScript Object, created by a standard JavaScript object constructor.

AngularJS Scope

The scope is the binding part between the HTML (view) and the JavaScript (controller).The scope is an object with the available properties and methods.The scope is available for both the view and the controller.

{{carname}}

The property "carname" was made in the controller, and can be referred to in the view by using the {{ }} brackets.

My name is {{name}}

When you change the name in the input field, the changes will affect the model, and it will also affect the name property in the controller.
